# API Monitor Backend

Spring Boot backend for an API Monitoring System.

## Features

- JWT Authentication
- CRUD for API Endpoints
- Monitoring Logs
- Dashboard Statistics
- API Status Tracking
- PostgreSQL Database
- Spring Security

## Tech Stack

- Java 17
- Spring Boot
- Spring Security
- JWT
- PostgreSQL
- Maven

## Run

1. Clone repository
2. Configure PostgreSQL
3. Update application.yml
4. Run

mvn spring-boot:run
